## Windows環境での動作確認手順:
1. Python 3.8+ がインストールされていることを確認
2. 必要な依存関係をインストール: pip install -r requirements.txt
3. 環境変数を設定:
   - HINEMOS_BASE_URL=http://3.113.245.85:8080/HinemosWeb/api
   - HINEMOS_USERNAME=hinemos
   - HINEMOS_PASSWORD=hinemos
4. FastMCPサーバを起動: python hinemos_fastmcp_server.py
